OperationEnum

Bus State enum.

NameNumberDescription
SAFE_MODE_BUS 0

Bus is in safe mode. Will not respond to commands from payload.
SUN_POINT 1

Bus is in sun pointing. Will not respond to commands from payload.
OPERATIONS_MODE 2

Bus is in operations mode. Will respond to commands from payload.
MAINTANCE 3

Bus is in maintance mode. Will respond to commands from payload.
GROUND_TEST 4

Bus is in ground test mode. Will respond to commands from payload.

PayloadEnum

Payload State enum.

NameNumberDescription
SAFE_MODE_PAYLOAD 0

Payload is in safe mode. All commmand fields will be ignored by bus.
IDLE 1

Payload is in idle mode. All commmand fields will be ignored by bus.
OPERATIONAL 2

Payload is in Operational Mode. Command fields will be acted upon by bus.
