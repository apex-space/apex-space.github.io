OperationEnum

Bus State enum.

NameNumberDescription
SAFE_MODE_BUS 0

Bus is in safe mode. Will not respond to commands from payload.
SUN_POINT 1

Bus is in sun pointing. Will not respond to commands from payload.
OPERATIONS_MODE_INTERTIAL_POINTING 2

Bus is in operations mode. Will respond to commands from payload.
OPERATIONS_MODE_DELTA_MANEUVER 3

Bus is in operations mode. Will respond to commands from payload.
MAINTANCE 4

Bus is in maintenance mode. Will respond to commands from payload.
GROUND_TEST 5

Bus is in ground test mode. Will respond to commands from payload.

PayloadEnum

Payload State enum.

NameNumberDescription
SAFE_MODE_PAYLOAD 0

Payload is in safe mode. All command fields will be ignored by bus.
IDLE 1

Payload is in idle mode. All command fields will be ignored by bus.
OPERATIONAL 2

Payload is in Operational Mode. Command fields will be acted upon by bus.

State Machine Diagram

The bus is commanded between states via either ground commands or automated sequences. Once in Operation mode, the bus will have a variety(SunAq, RateDamp,ect) of GNC guidance modes that it can transition to. In External Guidance, the bus will close the loop on the commands sent in the Payload2Bus message.
